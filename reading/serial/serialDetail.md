
# 请求连载文章详细信息

## URL
http://v3.wufazhuce.com:8000/api/serialcontent/${id}

## 参数
+ id    ————>     简略信息中的id字段, 而不是serial_id

## 请求示例
+ http://v3.wufazhuce.com:8000/api/serialcontent/194

## HTTP请求方式
GET

## 数据返回格式
JSON

##是否需要登录
否
